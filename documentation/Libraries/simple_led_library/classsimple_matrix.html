false for right to left, true for left to right

General function to send over a column-arrayed buffer. Can be scrolled or not, depending in preference.

Warning
This function should not be directly used if you want to scroll a buffer. Instead, the scrollBuffer() function should be used for that.
Parameters
*matAn uint8_t column-addressed array containing the data to be sent.
columnThe number of columns to be sent out (in most cases this would be the same as the size of the *mat array).
start_fromWhere the buffer will start at in the display.
scrollWhether the outputed display will be scrolled from right to left.
delThe delay between each frame if scroll is set to true.
